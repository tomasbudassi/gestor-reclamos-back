# API Backend - Gestion de Reclamos
##Endpoints

### GET - Reclamos (All)
```
/reclamos/todos?page=0&size=10
```

### GET - Reclamos by Id
```
/reclamos/{id}
```

### GET - Reclamos by Status
```
/reclamos/estado/nuevo?page=0&size=10
```

### GET - Reclamos by Empleado
```
/reclamos/empleado/1
```

### POST - Crear Reclamo
```
/reclamos
```

### UPDATE - Actualizar Reclamo
```
/reclamos/{id}
```
